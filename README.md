# Contentful Content Modeling CLI (`ccm`)

The **Contentful Content Modeling CLI** (`ccm`) tool allows you to easily create and manage content models for your company inside of contentful using code. It abstracts the Contentful CLI with business logic for easier content modeling.

---

## Getting Started

- Make sure you have Node.js installed.
- Create a folder named 'contentful-content-modeling-cli'.
- Clone this repo in there.
- In that folder, run 'npm install'.
- Run 'npm link' (if you get an error, you may have to run 'sudo npm link' and login with your admin password).
- Test it using 'ccm --help' from any terminal.
